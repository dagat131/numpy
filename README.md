# numpy
- 넘파이배열에서 모든 원소의 데이터 타입은 int32
- 넘파이에 실수데이터 저장시 디폴트 데이터 타입은 float64

```
c=np.array([[1,2],[3,4]])

print(c.sum(axis=0)) #열방향

print(c.sum(axis=1))#행방향
```
```
[4 6]
[3 7]
```


### Concat
- vstack : 배열을 열방향으로 결합
  np.concatenate((A,B), axis=0)

- hstack : 배열을 행방향으로 결함
- column_stack : 1차원 배열을 열벡터로 하는 2차원 배열을 만듬
